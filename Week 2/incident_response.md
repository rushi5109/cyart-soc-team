# Basic Incident Response

## Incident Lifecycle (NIST SP 800-61)

1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

## Example Scenario

Incident: Ransomware detected on production server.

Identification:
Wazuh generated alert for suspicious file encryption activity.

Containment:
Isolated the infected machine from the network.

Eradication:
Removed malicious executable and patched vulnerabilities.

Recovery:
Restored system from clean backup.

Lessons Learned:
Improved monitoring and patch management policy.
