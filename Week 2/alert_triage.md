# Alert Triage Practice

## Sample Alert

Alert ID: 002  
Description: Brute-force SSH Attempts  
Source IP: 192.168.1.100  
Priority: Medium  
Status: Open  

## Triage Steps

1. Check login logs.
2. Identify number of failed attempts.
3. Verify IP reputation via VirusTotal.
4. Check for successful login.

## IOC Validation

The source IP was checked against threat intelligence platforms.
The IP was flagged as suspicious activity.

Conclusion: Escalated for further investigation.
